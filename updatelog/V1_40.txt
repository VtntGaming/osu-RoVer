-- V1.40

This update include:
+ New interface UI, contain:
  + Settings
  + Map infomation
  + Map result
+ Added detail in Perfomance display (AimPS/StreamPS/AccPS)
+ Buffed Accuracy Perfomance, make it take part as 20% of the map.
+ Nerfed Aim/Stream Perfomance, only take part as 80% of the map.
+ Added another type of perfomance is Overall perfomance.
+ Added detail perfomance on the Perfomance display.
+ Updated perfomance/difficulty caculation
+ Changes to PS accuracy punishment (Because of Accuracy buff)
    [details="Click to see what changed to PS punishment. (Mostly nerfed)"]
   + Decrease 47% PS lost and 33% PS lost for HC if miss. (-68% / 0% HC Permanent)
   + Decrease 35% PS lost and 25% PS lost for HC if 50s. (-13% / +66% HC Permanent)
   + Decrease 53% PS lost and 50% PS lost for HC if 100s. (+60% / +300% HC Permanent)
   + Permanent PS lost take with a calculated amound of Standard PS list by 20%.
   + Increase 100% PS recover rate.
   [/details]
+ Custom beatmap is now back. But **only** available on [BetaPlace]( https://www.roblox.com/games/7216838953/osu-RoVer-BetaPlace)
+ Fixed Hit key UI on Right Hitzone (Mobile Mode)
+ Some changes to Slider movement. 
+ Sliders now have slider tick, depend on how long the slider is you will either gain combo or **lose**  all of it.
+ Beatmap choose now load a little bit faster than before.
+ Added profile page, you can check your and other people top plays, customize Profile picture and Profile background. Can be loaded by clicking on the leaderboard or type on the search bar.
+ Added custom difficulty color editor [spoiler](This maybe called an update instead of new feature because it appear on V1.39)[/spoiler]
